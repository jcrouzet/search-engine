Régionales : Guillaume Peltier désigné chef de file UMP dans le Centre
Le Monde |
| Par Alexandre Lemarié
L’UMP a tenté de finaliser, jeudi 16 avril, la désignation de ses chefs de file pour les élections régionales prévues en décembre.
La commission nationale d’investiture du parti examinait les candidatures dans les cinq régions (sur les treize existantes) où il n’avait pas encore pris de décision.

Dans la région Centre-Val-de-Loire, le sarkozyste (ex-FN) Guillaume Peltier a été désigné chef de file UMP, ont indiqué au Monde des participants à cette réunion.
Il l’a emporté face à l’ex-ministre Hervé Novelli.
Cela ne fait pas pour autant de M. Peltier la tête de liste officielle de la droite dans cette région car les négociations entre l’UMP et l’UDI se poursuivent.
Le président du parti centriste, Jean-Christophe Lagarde, veut notamment imposer Philippe Vigier comme tête de liste d’une liste d’union UMP-UDI dans cette région.

En revanche, le parti n’est pas parvenu à s’accorder sur un chef de file pour la région Aquitaine-Limousin-Poitou-Charentes.
La désignation de la juppéiste Virginie Calmels, qui paraissait actée, a finalement été reportée, selon un participant à la réunion.
Olivier Chartier, proche de Jean-Pierre Raffarin, et Jérôme Peyrat, proche de Nathalie Kosciusko-Morizet, sont également candidats.
En Provence-Alpes-Côte d'Azur (PACA), la candidature de Christian Estrosi a été, sans surprise, officialisée.
Le maire de Nice conduira, à la demande de Nicolas Sarkozy, la liste UMP dans cette région, a annoncé le parti de droite, mardi, dans un communiqué.
La commission d'investiture de l'UMP vient d'approuver Estrosi en chef de file en Paca.

Autres régions où l’on attendait la désignation d’un chef de file : en Languedoc-Roussillon-Midi-Pyrénées – où le jeu est très ouvert entre l’ex-député Bernard Carayon, le politologue Dominique Reynié ou le député Pierre Morel-A-L’Huissier – et en Corse.
Huit chef de file UMP ont été désignés en février : Valérie Pécresse pour l’Ile-de-France, Xavier Bertrand pour le Nord-Pas-de-Calais - Picardie, Bruno Retailleau pour les Pays de la Loire, Philippe Richert pour Alsace - Champagne-Ardenne - Lorraine, Marc Le Fur pour la Bretagne, Alain Joyandet pour la Bourgogne - Franche-Comté, et Laurent Wauquiez qui a été préféré à Michel Barnier pour l’Auvergne - Rhône-Alpes.
Un binôme de deux femmes députées a été choisi pour la Normandie : les députées Véronique Louwagie et Françoise Guégot.
